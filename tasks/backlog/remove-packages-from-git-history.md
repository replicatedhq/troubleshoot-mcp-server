## Description
Remove the `packages/` folder from git history and rewrite history to clean up accidentally committed build artifacts. This folder contains APK build artifacts that should never have been in version control.

## Background
- The `packages/` folder was accidentally committed starting from commit 6264227
- It contains binary build artifacts (APK files and APKINDEX.tar.gz)
- The folder is already in `.gitignore` but needs to be removed from history
- Affects commits from 6264227 through current HEAD (approximately 6 commits)

## Implementation Requirements

### IMPORTANT: Work directly on main branch
This task requires history rewrite and MUST be done on the main branch, not in a worktree or feature branch.

### Pre-implementation Checklist
1. Create backup branch: `git branch backup-before-packages-removal && git push origin backup-before-packages-removal`
2. Ensure clean working directory: `git status` must show nothing to commit
3. Document affected commits for verification

### Implementation Steps

#### Step 1: Install Required Tool
Choose one of these options:
- **Option A (Recommended)**: Install git-filter-repo
  ```bash
  pip install git-filter-repo
  ```
- **Option B**: Download BFG Repo-Cleaner
  ```bash
  wget https://repo1.maven.org/maven2/com/madgag/bfg/1.14.0/bfg-1.14.0.jar
  ```

#### Step 2: Remove packages/ from History
- **Using git-filter-repo**:
  ```bash
  git filter-repo --path packages/ --invert-paths --force
  ```
- **Using BFG**:
  ```bash
  java -jar bfg-1.14.0.jar --delete-folders packages --no-blob-protection
  git reflog expire --expire=now --all && git gc --prune=now --aggressive
  ```

#### Step 3: Verification
Execute ALL of these verification steps:
```bash
# Should return nothing - if it returns results, the removal failed
git log --all -- packages/

# Double-check no packages/ in any tree
git rev-list --all | xargs -I {} git ls-tree -r {} | grep packages/ || echo "Success: packages/ not found in history"

# Verify .gitignore still contains packages/
grep "packages/" .gitignore

# Test that build process still works
./scripts/build.sh

# Verify packages/ is regenerated but ignored
git status  # Should show packages/ as ignored, not untracked

# Run full test suite
uv run pytest
```

#### Step 4: Document Changes
```bash
# Save new commit history for reference
git log --oneline > post-rewrite-history.txt

# Note repository size reduction
du -sh .git > post-rewrite-size.txt
```
